Жаргон и разъяснения

Платформа будет запущена в 3 университетах SG в этом порядке NTU и SMU, затем NUS, затем другие.
Администратор, используемый в этом документе, представляет системного администратора (владельца платформы).
Владелец страницы может быть администратором или постоянным пользователем. Это термин, используемый для идентификации создателя страницы (например, User A create Page NTU Club, Пользователь A является владельцем сайта клуба NTU).
Все функции удаления - это мягкое удаление, данные все равно должны оставаться в базах данных.
Все функции обычного пользователя должны быть реплицированы для администратора (admin seniorsays).
